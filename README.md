### **üíª Technical Projects (with links to code/notebooks)**
### **JumpPro**
**Technologies:** Python, Pandas, Streamlit, SciPy, Matplotlib  
Development of a signal-processing pipeline and exploratory model to detect and quantify jump events in real time using IMU data.  
Completed in collaboration with the Swedish Beach Volleyball Team to support performance monitoring and training feedback.

---

### **Shoulder Load Monitor**
**Technologies:** React, Swift, Bluetooth Low Energy (BLE)  
Prototype application that parses and visualises accelerometer and gyroscope data from an IMU to monitor glenohumeral joint activity in real time.  
Designed to support clinicians, coaches, and athletes in understanding shoulder loading patterns during movement.

---

### **Performance Protocol**
**Technologies:** React, Swift, Core ML, VNDetectHumanBodyPoseRequest  
Performance and communication tool aimed at helping coaches, therapists, and medical staff translate biomechanical and movement insights into clear, actionable guidance for athletes.  
Integrates pose detection and lightweight machine learning models to support movement assessment and decision-making.
